## 1 · PURPOSE

The Archetype Library provides named professional archetypes used in the
`professional_archetype` section. Each archetype name must be:
- specific and evocative
- grounded in the client's chart (Saturn, MC, Mars patterns)
- not generic or generic-astrology language
- Darrow Code original naming

## 3 · ARCHETYPE NAMING RULES

Each archetype must:
- be a short, memorable name (2–4 words)
- name a recognizable human pattern, not a star sign
- come from the specific Saturn / MC / element / chart-shape combination
- feel like something a person would actually recognize about themselves
- not be copied from personality-test labels (MBTI, Enneagram, etc.) verbatim

✅ Good: "The Gray Cardinal", "The Deep Builder", "The Silent Architect"
❌ Bad: "INTJ", "Type 5", "Strong Saturn person"
