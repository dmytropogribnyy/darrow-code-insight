# Darrow Code Source Packs

**Status:** raw research area / docs-only.

This directory is reserved for approved Darrow Code source-pack appendices.

## Purpose

Source packs preserve:

- reviewed source matrices;
- source-quality notes;
- source-risk notes;
- useful extraction candidates;
- claim boundaries;
- future rule seeds.

## Hard rule

Source packs are **not** runtime instructions.

They must not be used directly as:

- prompt instructions;
- schema rules;
- PDF/template rules;
- provider implementation instructions;
- customer-facing prose;
- authorization to activate gated layers.
