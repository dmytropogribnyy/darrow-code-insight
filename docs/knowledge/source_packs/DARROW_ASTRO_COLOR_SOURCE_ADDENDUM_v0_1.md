## 0. Core rule

This addendum is **not** a production color dictionary.

Astrology-color sources are useful for:
- symbolic color motifs;
- visual / STYLE add-on inspiration;
- palette taxonomy;
- market/category awareness;
- identifying claims Darrow Code must avoid.

They must **not** be used to:
- generate “lucky colors”;
- claim a color heals, protects, attracts love/money, raises vibration, fixes mood, or changes destiny;
- assign a single “correct” color from Sun sign only;
- copy external color descriptions, palettes, report text, images, or tables wholesale;
- present zodiac colors as scientifically established.

Darrow Code default:

```text
Color = psychological-symbolic anchor, not fate, medicine, luck, protection, or energetic guarantee.
```

Recommended wording:

```text
“A deep blue anchor can symbolically support emotional coherence and quiet authority.”
```

Forbidden wording:

```text
“Blue is your lucky color and will heal anxiety / attract success / protect you.”
```

## 3. Source conflict / no-consensus note

Astrology-color sources do **not** agree perfectly.

Examples:
- Taurus can appear as green, pink, pastel, or earth-toned depending on whether the source emphasizes Venus, earth element, sensuality, nature, or softness.
- Libra can appear as blue, pink, pale green, lavender, or pastel depending on whether the source emphasizes balance, Venus, air, elegance, or calm.
- Pisces can appear as lavender, seafoam, green, blue-gray, or indigo depending on whether the source emphasizes Neptune, Jupiter, water, dream, or mysticism.
- Aquarius can be blue, turquoise, electric blue, neon, violet, white, or Saturnian black depending on whether the source emphasizes Uranus, air, water-bearer symbolism, or traditional Saturn rulership.

Darrow conclusion:

```text
Do not build a rigid “one sign = one color” production dictionary.
Build a convergence-based symbolic color layer:
planetary function + element + sign + dominant chart motif + report context.
```

## 4. Darrow color selection model — draft

Recommended future logic:

```text
1. Start with chart emphasis:
   - dominant planets
   - dominant elements
   - Sun/Moon/Ascendant only when reliable
   - major aspects
   - BaZi/Wu Xing convergence if available

2. Identify symbolic need:
   - grounding
   - visibility
   - calm
   - precision
   - warmth
   - boundary
   - renewal
   - depth

3. Select:
   - 1 primary anchor color
   - 1–2 supporting colors
   - optional “avoid overuse” note

4. Phrase as symbolic support, not destiny/luck/healing.
```

Example safe output:

```text
Your chart has a strong Saturn/Earth pattern, so charcoal and forest green can work as symbolic anchors: charcoal for boundary and seriousness, forest green for slow organic renewal. These are not lucky colors; they are visual reminders of the rhythm your system already understands.
```

## 8. Forbidden claims policy

The color layer must never say or imply:

```text
- this is your lucky color;
- this color will heal you;
- this color protects you;
- this color attracts love/money/success;
- this color raises your vibration;
- this color activates a chakra;
- this color makes you psychic;
- this color lowers blood pressure / changes physiology;
- you must wear this color;
- avoiding this color will harm you;
- every Aries/Taurus/etc. must use the same color.
```

Allowed language:

```text
symbolic anchor
visual reminder
palette direction
aesthetic mirror
soft support
psychological cue
report-style motif
```
