## 0. Core rule

Allowed:
- general symbolic traditions
- public facts and metadata
- public-domain / CC0 / properly licensed reference data
- cross-source synthesis
- our own Darrow Code interpretations
- internal source log

- copying astrology articles/reports/text
- lightly paraphrasing copyrighted interpretations
- scraping private-use-only or restricted datasets
- copying protected tables/databases wholesale
- adding Astro.com/Astrodienst/Swiss Ephemeris PDFs/files into repo
- presenting symbolic systems as scientific fact

| Name origin / language | Name metadata | Cultural/etymological note | Avoid claims if uncertain |
| Name archetype | Meaning + numerology convergence | Optional identity-card line | Must be clearly symbolic |
| Transliteration variants | Non-Latin names | Future normalization | Must preserve user identity |

### 4.2 Name-data policy

Use name etymology as a soft symbolic layer only.

Allowed wording:
- “Your name layer adds a symbolic note of…”
- “Etymologically, this name is often connected with…”
- “As a symbolic mirror, the name points toward…”

Avoid:
- “Your name proves…”
- “This name guarantees…”
- “Because of your name, you are…”
- “Your name means you will…”

## A0. Appendix rule

These sources are **not** primary etymology authorities.

They may be useful for:

- astrology-inspired naming trends;
- celestial name motifs;
- Russian-language name catalogues and spelling variants;
- Russian esoteric name-symbolism market awareness;
- examples of deterministic claims Darrow Code must avoid.

They must not be used to:

- copy name lists wholesale;
- copy name descriptions or interpretive paragraphs;
- scrape protected catalogues/databases;
- claim that a name determines destiny, personality, health, love, success, wealth, fertility, luck, or spiritual status;
- recommend that a user change their name;
- assign a name by zodiac sign as if it were objectively correct.

Darrow Code rule:

```text
Name symbolism is a secondary mirror.

Name etymology must be verified before use.
Lifestyle / baby-name / esoteric sources are inspiration-only unless cross-checked.
```

## A5. Name-source confidence policy

Name layer confidence should be stored as:

| Confidence | Meaning | Use |
|---|---|---|
| high | verified across multiple etymology / onomastics sources or open metadata | may be used as soft identity note |
| medium | common meaning appears in several popular sources, but scholarly origin is uncertain | use carefully with “often connected with…” |
| low | lifestyle / baby-name / esoteric source only | inspiration only; do not use as factual claim |
| blocked | deterministic, medical, success, luck, compatibility, or name-change claim | do not use in customer report |

Rules:

1. Do not use lifestyle baby-name sources as etymology authorities.
2. Do not generate name destiny claims.
3. Do not recommend changing a name.
4. Do not claim a name causes success, failure, love, health, wealth, fertility, personality, or luck.
5. Use celestial / astrology-name sources only for optional symbolic mirrors.
6. Chart and numerology remain primary.
7. Name symbolism is secondary and must be clearly framed as symbolic.
