## 0. Core rule

This layer must be treated as a **modern symbolic mirror**, not as a historically certain ancient Celtic personality system.

Allowed:
- tree symbolism as poetic / ecological / archetypal mirror;
- general public folklore motifs if sourced carefully;
- horticultural and botanical facts as factual background;
- modern Celtic Tree Zodiac tables as **popular tradition reference**;
- Darrow Code original interpretations.

Not allowed:
- claiming the 13-sign tree zodiac is proven ancient Druid science;
- deterministic birth-tree claims;
- compatibility claims;
- healing / protection / fortune / ritual guarantees;
- gemstone/color/spirit-animal recommendations as factual or magical necessities;
- copying source prose or tables wholesale;
- turning this into CORE runtime before a curated dictionary is approved.

Darrow Code wording rule:

```text
Use as “modern symbolic mirror,” “tree archetype,” “nature-symbolic reflection,” or “poetic mirror.”
Do not use as “ancient verified Celtic astrology” or “your true Druid sign determines...”
```

## 2. Key source observations

### 2.1 The common popular structure

Across popular Celtic Tree Astrology sources, the recurring structure is:

- 13 lunar-style periods;
- each period mapped to a tree or plant;
- a nickname/archetype;
- personality traits;
- sometimes compatibility;
- sometimes additional correspondences such as animals, colors, gemstones, gods, or Ogham letters.

Darrow Code may use the **tree motif** and the **symbolic archetype** as raw inspiration, but should not import compatibility, magical guarantees, or correspondence stacks.

## 3. Historical caution note

Modern Celtic Tree Astrology should not be presented as a cleanly verified ancient Druidic system.

Important cautions:
- Popular articles often claim ancient Druid origins, lunar months, and Ogham correspondences.
- Ogham/tree associations are historically complex and not all letter names are originally tree names.
- Some modern “tree calendar” systems are associated with modern revival / literary-esoteric reconstruction, not direct ancient documentation.
- Several popular sources add animals, gemstones, colors, gods, compatibility, and “energy” claims; these should remain blocked.

Darrow Code position:

```text
Tree symbolism can be used as a poetic, nature-based mirror.
It should not be framed as a historically certain ancient Celtic personality science.
```

## 5. What to block

### 5.1 Compatibility

Several popular sources include tree-sign compatibility pairings. Darrow Code should not use them.

Blocked:

```text
Oak is compatible with Ash and Reed.
Willow pairs best with Ivy and Birch.
```

Allowed:

```text
Tree symbolism can describe relationship tendencies only when supported by chart/numerology data.
```

### 5.2 Magical correspondences

Popular sources may include:
- gemstones;
- colors;
- spirit animals;
- Celtic gods;
- good fortune;
- protection;
- “energy” strengthening;
- meditation under a birth tree.

These are **blocked for CORE** unless a curated symbolic dictionary is explicitly approved.

Allowed soft version:

```text
Oak can be used as a symbolic image of rooted strength.
```

Blocked version:

```text
Wear oak jewelry or meditate under oak to draw strength and protection.
```
