## 12 · MODULE SAFETY RULES

- **MONEY / money_and_value:** no wealth promises, no financial advice.
- **BODY / vitality_baseline:** no medical claims, diagnosis, cure, or healing;
  carry the disclaimer.
- **YEAR / current cycle (in executive_summary):** orientation and timing climate
  only; no deterministic prediction of events.

  protection claims.

  no specific cities, no "best places."

  compatibility or attraction.

## 13 · PROOF TAG RULES

- Maximum **5 proof tags per section**.
- Real, available data only — no invented placements.
- No tags for unavailable data.
- If `birth_time_known=false`: no house or angle tags.
- Proof tags are confirmation, placed at the end — never the content.

## 14 · PROTOCOL / WARNING SIGNAL RULES
